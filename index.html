2. Project Overview

Smarter irrigation, healthier crops

Smart AgroTech monitors soil moisture and controls irrigation based on real-time conditions. The goal is to avoid over-watering and under-watering by combining IoT sensors, a microcontroller, and a user-friendly dashboard.

Water Efficiency

Only irrigate when needed, reducing water usage and improving sustainability.

Better Decisions

Use live readings to decide irrigation timing instead of fixed schedules.

IoT Ready

Scalable architecture: sensors → controller → cloud/dashboard → control.

3. Problem Statement

Traditional irrigation wastes water

Many farms use manual irrigation or fixed schedules, which do not consider changing soil conditions. This leads to water wastage, uneven crop growth, higher electricity consumption, and unnecessary labor.

Key issues

- Over-watering and under-watering
- Lack of real-time monitoring
- Manual workload and delayed decisions
- Electricity usage due to inefficient pumping

Impact

- Reduced yield due to inconsistent moisture
- Soil nutrient loss (leaching) from excess water
- Increased operational costs for farmers
- Unsustainable water management

4. Proposed Solution

Automate irrigation using IoT

The system collects soil moisture data, compares it with a threshold, and then turns the pump ON/OFF accordingly. A dashboard displays readings and allows manual override.

Sense

Read soil moisture via sensor and convert to a usable percentage.

Decide

Compare values with threshold and decide pump action automatically.

Act

Control the pump/relay and show status in the dashboard for users.

8. Working / Methodology

Step-by-step workflow

How the system behaves from sensing to irrigation control and monitoring.

1

Sense moisture

Sensor measures soil moisture continuously and sends readings to controller.

2

Convert & filter

Controller converts raw sensor value into percentage and smooths noisy data.

3

Apply decision logic

Compare moisture value with threshold to decide whether irrigation is required.

4

Control pump via relay

Relay activates pump for watering or turns it OFF when moisture becomes enough.

5

Update dashboard

Dashboard shows readings, pump status, and logs for monitoring and demo.

9. Algorithm (Simple Steps)

Easy to understand logic

Beginner-friendly algorithm description for reports and presentations.

Algorithm steps

1. Read soil moisture sensor value
2. Convert to moisture percentage
3. If moisture < threshold → turn pump ON
4. Else → turn pump OFF
5. Display moisture & pump status on dashboard
6. Repeat at fixed interval

Notes

- Add a small delay to avoid frequent switching (optional)
- Allow manual override from UI
- Log changes for monitoring and debugging
